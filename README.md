# 🎮 HTML 테트리스

브라우저에서 바로 플레이할 수 있는 **HTML/CSS/JavaScript 테트리스 게임**입니다.  
별도의 설치 없이 HTML 파일만으로 실행 가능하며, 전통적인 테트리스 룰과 다음 기능을 제공합니다.

---

## 🟢 주요 기능

- 빨강/노랑 블록으로 색 구분
- 다음 도형 미리보기
- HOLD 기능 (`C` 키)
- 라인 삭제 시 화면 흔들림 효과
- 블록 사이 여백으로 날렵한 스타일 구현
- 정통 테트리스 도형 및 룰 적용
